Keyboard input:
W, A, S, D, E, X: moves camera pos
SPACEBAR: spawns new random item(cylinder,sphere or cube)
<, >: changes the speed of spawned items
UP, DOWN, LEFT, RIGHT (arrow keys), +, -: moves the large red sphere
